# Ideas (Non-Binding)

Scratchpad for **exploratory design notes** that are **not** accepted decisions.

| Rule | Detail |
|------|--------|
| **Status** | Always **Idea** — not Implemented, not Accepted |
| **README link** | One row under **Where to go next** / **次に読むもの** only — label **not adopted / out of backlog**; remove row if idea closed |
| **Promotion** | Copy to `docs/decisions/TECH_STACK_*.md` and set **Proposed → Accepted** when agreed |
| **Implementation** | No code obligation until promoted |
| **Language** | English (logic docs policy) |
